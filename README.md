# Landrix - Commercial Landscaping & Snow Removal Template

Landrix is a premium, production-ready HTML template designed for modern commercial landscaping and snow removal businesses. It features a robust layout, seasonal toggling, and role-based dashboards.

## Features
- **Seasonal Toggle:** Switch between Summer (Landscaping) and Winter (Snow Removal) modes.
- **Dashboards:** Admin, Client, and Crew dashboards included.
- **RTL Support:** Full Right-to-Left support for Arabic/Hebrew.
- **Dark Mode:** System preference detection and manual toggle.
- **Responsive:** Mobile-first design using Tailwind CSS.

## Installation
1.  Clone or download the repository.
2.  Open `pages/index.html` in your browser.
3.  (Optional) Install Tailwind CSS standalone to recompile styles if editing `input.css`.

## Structure
- `assets/`: CSS, JS, Images, Fonts
- `pages/`: HTML pages (Public, Auth, Dashboard)
- `documentation/`: Detailed guide
